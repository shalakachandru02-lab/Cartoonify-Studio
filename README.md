# 🎨 Cartoonify Studio

A Python-based image cartoonification application built using OpenCV, NumPy, Matplotlib, Pillow, and Tkinter. The application allows users to transform images into multiple artistic styles through an interactive graphical user interface.

## Features

* Anime Style
* Watercolor Style
* Realistic Painting
* Comic Effect
* Pop Art
* Interactive Tkinter GUI
* Image Preview and Save Functionality

## Technologies Used

* Python
* OpenCV (cv2)
* NumPy
* Matplotlib
* Pillow (PIL)
* Tkinter

## Installation

Clone the repository:

```bash
git clone https://github.com/sharanya-gif/Cartoonify-Studio.git
```

Navigate to the project folder:

```bash
cd Cartoonify-Studio
```

Install dependencies:

```bash
pip install opencv-python numpy matplotlib pillow

```

```bash
pip install -r requirements.txt


## Run the Application

```bash
python cartoonn.py
```

## How It Works

1. Launch the application.
2. Select an image from your computer.
3. Choose a style:

   * Anime
   * Watercolor
   * Realistic Painting
   * Comic Effect
   * Pop Art
4. Click "Choose Image & Cartoonify".
5. View and save the generated image.
